## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`

2. Run the app:
   `npm run dev`

3. Open the app in your browser (usually at `http://localhost:3000`), then enable mobile view in DevTools:
   - **Chrome / Edge**: `F12` → click the **Toggle device toolbar** icon (or `Ctrl+Shift+M`) → select a device or set a custom resolution
   - **Firefox**: `F12` → click the **Responsive Design Mode** icon (or `Ctrl+Shift+M`) → select a device or set a custom resolution
   - **Safari**: Enable DevTools first via *Settings → Advanced → Show Develop menu*, then *Develop → Enter Responsive Design Mode*
